# Ensemble des modules Python créés pour le stage
### Decription
Ce dossier contient les trois modules Python que l'on utilise au cours du stage.
- noise.py : création et application de bruit additif
- metrics.py : mesures liés au signal
- tools.py : outils de débruitage et outils de lecture de fichier

### Remarques
Certains modules contiennent des fonctions nécessitant une carte graphique GPU dédiée et le téléchargement de données supplémentaires.
