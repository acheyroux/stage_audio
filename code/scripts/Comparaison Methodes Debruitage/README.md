# Script de Comparaison de Méthodes de Débruitage
### Contexte
Prend des données de SNR de sortie pour divers méthodes et trace un graphe de comparaison.

### Instructions d'utilisation
1. Copier-coller les résultats de chaque expérience dans leurs fichiers respectifs de ```/SNR Data/``` sans changer de format
2. Ouvrir le fichier ```params.txt``` pour confirmer les paramètres ou les modifier dans le cas échéant
3. Lancer le script a l'aide de la commande ```python experience_sgmse.py``` dans l'environnement (audio-denoising) de Miniforge Prompt
4. Recueillir les données
